# Ruribitaki
Ruribitaki is the twitter library for vala language.

## Description
Ruribitakiは,valaで使用することのできるTwitterライブラリです.  
現在,アカウント情報の更新,及びダイレクトメッセージ関連のAPIには対応していません.

## Install
    $ ./autogen.sh
    $ ./configure --prefix=/usr
    $ make
    $ sudo make install
